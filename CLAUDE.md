## Prompt Changelog

When the user makes a feature suggestion or request (not a bug fix), append a timestamped 1-sentence summary to `CHANGELOG-PROMPTS.md` in this format:
```
- **YYYY-MM-DD HH:MM** — New: <1-sentence summary of the feature request>
  - *"<the user's actual prompt, lightly trimmed if very long>"*
```
Do this silently — no need to mention it to the user.
